# Hiraeth

A choose-your-own-adventure game made in pure Python for the terminal/CMD, created as my final project for CS50 Python.

## Features

* Interactive story with player choices
* Multiple paths and endings
* ASCII art in the terminal
* Built entirely with Python3
* Runs in CMD/terminal

---

## Run the Project

Clone the repository:

```bash id="kth7o8"
git clone https://github.com/notnada/Hiraeth.git
cd hiraeth
```

Run the game:

```bash id="v9s2he"
python3 Hiraeth.py
```

## About

The game focuses on storytelling, atmosphere, and decision-making through a simple terminal experience.

The name *Hiraeth* comes from a Welsh word meaning a deep feeling of longing or nostalgia for a place/home that never existed.
